XTRM_TAXINFO

Purpose

This is a sample program, intended to demonstrate the XTRM™ API for tax information. As tax information can contain personal and identifying information as defined by GDPR and similar standards, allowing anyone other than the taxpayer or taxpaying entity to access this information is problematic.

XTRM™’s solution is to email the tax form, as a PDF, directly to the registered email of the taxpayer. The API allows for a company to send tax information, without themselves being privy to any information on that tax form.

Usage

xtrm-taxinfo

--debug, -d optional, default false

Enable debugging and other error messages.

--manager, -m mandatory

The SPN number of the company requesting tax information to be sent

--recipient, -r mandatory

The PAT or SPN number of the company to recieve tax information

--taxyear, -t mandatory

A four-digit year identifying the tax year for which to send tax information

--verbose, -b optional, default false

Enable descriptions and other messaging.

A typical invocation might be:

xtrm_taxinfo ‑‑taxyear 2020 ‑‑recipient AT12121234 ‑‑manager SPN89898912

Implementation Quirks

Usage of this program requires an API key and API secret for authentication. Secrets may be stored in database, in an .ini file, or in various other ways. This program has a dummy configuration which embeds the secrets directly into the code. The developer will need to make changes only to configSecrets.go to return the secrets to the rest of the program, and may choose whatever level of security seems appropriate for the project.

Downloading and running the program

As a Go program, this program may be downloaded via github, and run or installed with go run or go install executed from the code directory.

The specific logic of calling the tax information API is contained in xTaxRequest.go.
